Anthropic API Key
スクリーンショットからスコアを自動読み取りするために使用します。
APIキーは console.anthropic.com で取得できます。
入力したキーはこのブラウザのlocalStorageにのみ保存され、外部には送信されません。
楽曲データフォーマット
以下のJSON形式のファイルをSONGSタブの「楽曲データ読み込み」から読み込めます。
同じtitle＋difficultyの組み合わせは上書き更新されます。
[ { "title": "Tempestissimo", "artist": "t+pazolite", "difficulty": "FTR", "constant": 11.7, "totalNotes": 1403 }, { "title": "Testify", "artist": "void (Mournfinale) feat. 星熊南巫", "difficulty": "BYD", "constant": 12.0, "totalNotes": 2221 } ]
・ artist: アーティスト名（省略可）
・ difficulty: FTR / ETR / BYD のいずれか（対象外のPST/PRSは読み込まれません）
・ constant: 譜面定数（例: 11.7）
・ totalNotes: 総ノーツ数（整数）
データ管理
スコアデータのバックアップ・別端末への移行に使用します。楽曲定数データは別途JSONで読み込んでください。
